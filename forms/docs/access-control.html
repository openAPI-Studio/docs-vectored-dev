Access Control

Control who can view and submit your forms with flexible access levels.

Access Levels

Choose who can access your form:

  • Anonymous — anyone with the link can submit (no login required)
  • Logged-in users — only authenticated Confluence users
  • Specific groups — restrict to selected Confluence groups

Single Submission Limit

Enable the single-submission toggle to allow each user only one response. Subsequent attempts show a lock screen indicating the form has already been completed.

Lock Screens

Unauthorized users or those who have exceeded submission limits see a lock screen explaining why they cannot access the form.
